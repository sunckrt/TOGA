FCOM 2.70.10 — Engine — Abnormal and Emergency Procedures — Rev. 47

2.70.10.1 ENGINE FIRE DETECTION SYSTEM — The aircraft is equipped with continuous loop fire/overheat detection systems for each engine. Each loop consists of sensing elements located within the engine nacelle. The control module provides a discrete warning signal when both loops detect a fire or overheat condition simultaneously, preventing nuisance warnings.

2.70.10.2 WARNINGS AND INDICATIONS — When fire is detected, the following indications occur simultaneously: The FIRE pushbutton on the affected engine illuminates red. The MASTER WARN light flashes. The continuous repetitive chime (CRC) sounds. The ECAM displays the E/WD warning in red with the associated procedure. The ENG FIRE warning is classified as a Level 3 Warning.

2.70.10.3 THRUST LEVER — The affected engine thrust lever must be moved to the IDLE detent as a first action to reduce fuel and hydraulic fluid flow to the nacelle. This action must be accomplished before the ENG MASTER is set to OFF to ensure correct sequencing of the procedure and avoid uncommanded engine shutdown.

2.70.10.4 AGENT DISCHARGE — Two extinguishing agents (AGENT 1 and AGENT 2) are available per engine. AGENT 1 should be discharged first. If the fire warning persists after 30 seconds, AGENT 2 may be discharged. After both agents have been used, no further extinguishing capability is available.

2.70.10.5 POSTFIRE CHECKS — Following confirmed engine fire extinction, the crew should monitor for rekindling, observe EGT trends, and consider diversion to the nearest suitable aerodrome. The APU may be started as applicable to supplement electrical and pneumatic power from the single operating engine.
